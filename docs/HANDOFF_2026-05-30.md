## 3. ★ 다음 할 일 (PENDING — 최우선)
**회전율 축소 테스트**: 신호 강할 때만 매매(|q50| threshold 또는 보유기간 연장)해서 385회 → 수십 회로 줄였을 때,
**시초가 체결 + 수수료(1bp) + 세금(매도건별 15.4%) 후에도 B&H(+8.3%)를 이기는지** 확인.
- 이게 수수료+세금을 **동시에** 공격하는 유일한 레버 (회전 줄면 둘 다 ~10배 감소).
- threshold τ grid (예: |q50|>0, 0.3, 0.5, 0.75, 1.0 bp) 별로: 매매수, 시초가 누적(무비용/1bp/1bp+세금), vs B&H 비교 → 같은 형식 그림.
- 데이터: `predictions_xgb_v3_intervals.csv`(fold3) + `live_oos_2026_xgb.csv` (q50, y_true) + `data/raw/kosef_10y_daily_2023_2026.csv`(조정가 OHLC, etf_o2c=close/open-1).
- 포지션 = -sign(q50), |q50|>τ 일 때만. 세금 = tax_per_day(매도건별, 이익 트레이드만 15.4%, 손실 상계 X).
- 결과 해석: 회복되면 "저빈도면 가능"(강한 결론) / 안 되면 "robust하게 어렵다"(현 결론 확정).

**(선택) KTB 선물 비교**: 비용 ~0.5bp + 세금 양도소득 10%·손실통산·250만원 공제 → 배당 15.4% 무상계보다 훨씬 유리. 세제만 바꿔도 결과 달라질 수 있음 (데이터 접근은 학부 범위 밖일 수 있음 — Path C 기록 참조).

## 4. 발표 결론 framing (테스트 후 확정)
> "모델의 1일 방향 예측력은 통계적으로 입증(DM 유의). 그러나 현실 체결(시초가)·수수료·**특히 채권ETF 매도 건별 세금**까지 반영하면 고빈도 현물 매매로는 B&H 대비 우위가 거의 사라진다. 병목은 예측력이 아니라 **회전율×세금 구조**이며, 저빈도 전략 또는 KTB 선물(유리한 세제)이 후속 과제."

차별점 = ① 1일 방향 엣지(간밤 US→KR 스필오버) ② 정직한 방법론(누수 3회·가설 기각·체결현실성·분배금버그·비용/세금 규명). "예측 정확도 ≠ 거래 수익성" 자체가 강한 학술 메시지.
