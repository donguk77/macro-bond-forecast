## v3 → v4 (2026-04-30)

### 변경 배경
3인 다각도 외부 리뷰(학술 / 채권 도메인 / 학부 조교)로 v3 의 잔존 약점이 발견됨. 세 리뷰어가 동시에 critical 한 결함을 지적한 부분은 없으나(방향성·목적은 모두 인정), 각자의 시각에서:
- **학술 리뷰어**: exchangeability·DM 보정·분위수 폭 SHAP 등 정밀화 권고
- **도메인 리뷰어**: 변수 누락(DXY/이벤트 더미/스프레드/외인잔고) + 위기구간 정량 정의
- **학부 조교**: "**욕심 부리고 있음**" — 5주차 과부하·CQR 학습 부담·1주차 SHAP-LSTM 검증·"잠정 freeze" 함정·Streamlit 첫 시도 등

세 시각이 충돌하는 지점은 학부 조교 의견을 가장 무게 있게 받되(채점·학생 보호), 도메인의 저비용 추가와 학술의 표현 수정만 선별 채택 — 결과적으로 단순화 + 저비용 보강 조합으로 v4 패치.

### 변경 요약
1. **단순화** (학부 조교 채택): TCN 삭제, Optuna→수동 grid 5×5, CQR 선택 강등, SHAP-LSTM 검증 4주차로 이동(1주차는 XGBoost+TreeExplainer), Streamlit→Jupyter notebook 메인, 변수 freeze 1주차 말로 앞당김.
2. **저비용 보강** (도메인 채택): DXY 추가, 이벤트 더미 4개(FOMC/금통위/美 NFP/美 CPI ±1일), 위기구간 정량 정의(rolling vol 상위 20% ∪ 이벤트), YTM/지표금리 명시, 만기 갱신 처리 옵션.
3. **표현 정밀화** (학술 채택): 분위수 폭 SHAP → 각 분위수 SHAP 비교, DM test HLN+Bonferroni 보정, CQR marginal coverage 점근성 면책, 발표일 시프트의 변수별 차이 명시.
4. **보류** (욕심): NEX-CP/EnbPI/ACI, 외국인 채권 보유잔고, 한미 스프레드(EDA에서만), Walk-forward 3 fold + Optuna 결합.
