# 한국 국채 10Y 일일 예측 대시보드 — 설계서 v2

> **하나의 Δy 예측 → 이중 co-primary 출력**: ① 방향/거래 신호 ② 불확실성 구간
> 작성 2026-05-30 (v2: 이중출력으로 재설계) · 상태: 설계확정·구현대기 · 스택: Streamlit

---

## 1. 제품 포지셔닝 (정직)

**핵심:** 한국 10Y 국채 금리의 **내일 변화(Δy)** 를 매일 예측하고, 그 예측을 **두 가지로** 읽어주는 의사결정 보조 대시보드.

- **부호 sign(q50)** → 방향/거래 신호 (어느 쪽으로 움직일까)
- **범위 [q05, q95]** → 불확실성 구간 = 내일 P&L 분포 / VaR (얼마나 움직일까)

같은 한 번의 예측을 두 얼굴로 보여준다. 양자택일이 아니라 **둘 다 co-primary**.

**무엇이 아닌가:** 수익을 보장하는 자동매매 블랙박스 ❌. 두 출력 모두 **작동 구간과 한계를 라벨과 함께** 제시한다(아래 §6 정직성 가드).

## 3. 정보 구조 (이중 co-primary)

1. **[히어로 좌] 방향/거래 신호** — sign(q50) ▲/▼, 신뢰 국면(추세장 여부), 누적손익(1bp 비용)
2. **[히어로 우] 불확실성 구간** — online conformal 90% 구간, VaR 환산, 현재 변동성 레짐
3. **커버리지 점검** (롤링) — 목표 0.90 vs 실제. 구간 신뢰도의 핵심 정직 패널
4. **간밤 글로벌 동인** — US10Y·VIX·DXY 변화 → 한국 금리 압력
5. **정직한 성능 고지** — in-sample vs 2026 라이브 OOS 수치 + 한계(상시 표기)

## 5. 기술 아키텍처

**스택:** Streamlit (`streamlit run app.py`).

**오프라인(freeze):** `scripts/38_live_oos_2026.py` 레시피로 v3 분위수 모델(q05/q50/q95) 학습(train≤2024, val 2025 early-stop), cal 2025H2 conformity로 온라인 conformal 버퍼 초기화 → 모델·스케일러·버퍼 직렬화(`reports/model_v3_frozen.pkl`).

**서빙:** `features_v3_candidate.csv` **마지막 완전 행**으로 다음 영업일 q05/q50/q95 예측 → 최신 conformal Q 적용 → 구간 + 방향. 손익·커버리지는 `live_oos_2026_xgb.csv` + 롤링.

**갱신:** `02_collect_data.py` → `37_regen_features_2026.py` → freeze 재실행 → 대시보드. MVP는 수동.

## 6. 정직성 가드레일 (필수)

- 누적손익 곡선은 **항상 비용반영 + CI**, "보장 수익" 표현 금지.
- 방향 신호엔 **"추세장 한정·라이브 단독 유의성 없음·2bp면 상쇄"** 라벨 상시.
- 구간엔 **"위기 첫날 과소·외삽 한계"** 라벨, 커버리지 미달도 그대로 표기.
- 성능 고지 패널은 접기 가능하나 제거 불가.

## 7. ⚠️ 알려진 한계 (대시보드에 명시 + 발표 한계 슬라이드 재료)

**XGBoost 외삽 불가 → 위기구간 과소대응 (실측 증명):**

| 구간 | 고정 CQR | 온라인 conformal |
|---|---|---|
| 전체 | 0.872 | 0.905 |
| fold2(2021–22 레짐전환) | **0.672** | 0.879 |
| VIX 고변동 | 0.797 | 0.889 |
| VIX 저변동 | 0.948 | 0.921 |

- 트리 모델은 학습 잎 값 범위 밖을 외삽 못 함 → 전례 없는 위기에 분위수가 너무 좁아져 커버리지 붕괴(고정 CQR fold2 0.67).
- **온라인 conformal로 완화**(0.67→0.88): 최근 실현오차로 밴드를 적응적으로 넓힘. 단 **위기 첫날은 시차로 못 막고**, 완화 후에도 고변동(0.889)<저변동(0.921).
- → **T2(레짐 전용 모델) 향후과제**의 직접 동기. 대시보드는 이 한계를 숨기지 않고 "위기 첫날 신뢰 낮음" 경고로 노출.

**기타:** 데이터 지연(DXY ~1주) → as-of 날짜·신선도 경고 표기. 회전율 ~118%로 거래신호 비용 민감.

## 8. 확정된 결정 (v1 열린질문 해소)

1. **VaR 환산** — bp 구간 기본 + 듀레이션(D=8) 환산 **토글** 제공.
2. **다음 영업일** — **마지막 완전행 기준**(정직·재현 가능). 부분행 ffill 안 함.
3. **방향 신호** — **유지·co-primary**. 한계 라벨 상시.
4. **배포** — MVP는 **로컬 시연**, 여력 시 Streamlit Cloud.
